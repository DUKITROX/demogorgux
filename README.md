# Demogorgux

AI agent for demos — frontend web app.

## Run the app

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## What’s included

- **Homepage** — Title “Homepage”, URL input (“paste the URL here”), and “Launch demo” button. Submit or click to go to the demo view (optionally with the URL in state).
- **Demo view** — Video-conference style layout: main area (placeholder for future screen sharing) and a right-side chat where you can type messages to the AI agent. “Back” in the header returns to the homepage.

Built with React, Vite, and React Router.
